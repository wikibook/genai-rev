다른 번역 도구와 교차 검증하기

이번 영상에서는 구글 독스를 활용하여 다른 번역 도구와 교차 검증하는 방법을 살펴보겠습니다.

1. 구글 드라이브로 이동 (https://drive.google.com)

주소창에 구글 드라이브 주소를 입력해 구글 드라이브로 이동합니다.

2. [신규] 클릭

먼저 새로운 구글 스프레드시트를 생성하겠습니다. 왼쪽 메뉴에서 신규 버튼을 클릭하고

3. [Google 스프레드시트] 클릭

구글 스프레드시트 메뉴를 클릭합니다.

4. 제목에 '번역 검증 문서' 입력

새로운 구글 스프레드시트가 열리면 상단 제목 영역에 번역 검증 문서라고 제목을 입력합니다.

5. 첫 번째 행에 원문, 구글 번역, DeepL 번역, ChatGPT 번역 입력

첫 번째 행에 원문, 구글 번역, DeepL 번역, ChatGPT 번역을 입력합니다.

6. 첫 번째 행에 테두리 추가

첫 번째 행을 블록으로 선택한 다음 테두리 버튼을 클릭해 테두리를 설정합니다.

7. 첫 번째 행 가운데 정렬

이어서 가운데 정렬 버튼을 클릭해 가운데 정렬을 하겠습니다.

8. 첫 번째 행 배경색 지정

마지막으로 배경색을 지정합니다. 배경색 아이콘을 클릭하고 원하는 색상을 선택합니다.

9. B2 셀에 함수 입력

먼저 구글 번역을 위한 함수를 입력하겠습니다. B2 셀을 선택하고 다음과 같이 함수를 입력합니다.

10. 테두리 설정

두 번째 행에도 테두리와 정렬을 지정하겠습니다. 두 번째 행을 블록으로 선택한 후 테두리 설정 버튼을 클릭해 테두리를 설정합니다.

11. 줄바꿈 설정

이어서 줄바꿈 버튼을 클릭해 줄바꿈 설정을 합니다.

12. 상단 정렬 설정

마지막으로 세로 정렬을 상단 정렬로 설정합니다.

13. A2 셀에 원문 입력

구글 번역을 테스트 해 보겠습니다. A2 셀에 번역하고자 하는 원문을 입력합니다.

14. B2 셀에서 번역 결과 확인

잠시 기다리면 B2 셀에 구글 번역 결과가 나오는 모습을 볼 수 있습니다.

15. [확장 프로그램] 클릭

이어서 DeepL 번역을 설정해 보겠습니다. DeepL 번역을 설정하려면 DeepL API 키가 필요합니다. 아직 키를 발급받지 않았다면 DeepL API 계정 등록하기 튜토리얼을 참고하여 디벨 API 키를 먼저 발급받기 바랍니다. 상단 메뉴에서 확장 프로그램을 클릭하고.

16. [Apps Script] 클릭

Apps Script를 클릭합니다.

17. 기존 코드 지우기

화면에 있는 기존 코드는 모두 지워줍니다.
