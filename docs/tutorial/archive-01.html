GPT for Sheets and Docs 설치하기

이번 영상에서는 GPT for Sheets and Docs의 설치 방법을 살펴보겠습니다

1. GPT for Work 사이트로 이동 (https://gptforwork.com/)

브라우저 주소창에 GPT for Work 주소를 입력해 GPT for Work 홈페이지로 이동합니다

2. [Install] 클릭

화면 오른쪽 상단에 있는 Install 버튼을 클릭하고

3. [GPT for Sheets and Docs] 클릭

GPT for Sheets and Docs 메뉴를 클릭합니다

4. [설치] 클릭

구글 워크스페이스 마켓 플레이스 창이 나오면 설치 버튼을 클릭합니다

5. [계속] 클릭

설치 가능 창이 나오면 계속 버튼을 클릭합니다

6. 구글 계정 선택

계정 선택 창이 나오면 구글 계정을 선택하고

7. [허용] 클릭

이어서 허용 버튼을 클릭합니다

8. [완료] 클릭

설치가 완료됐다는 창이 나오면 완료 버튼을 클릭합니다

지금까지 GPT for Sheets and Docs의 설치 방법을 살펴봤습니다
