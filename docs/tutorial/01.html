DeepL 등록하기

이번 영상에서는 번역 서비스인 DeepL 계정을 등록하고 본격적으로 번역 서비스를 활용해 보겠습니다

1. DeepL 사이트로 이동 (https://www.deepl.com/)

브라우저 주소창에 DeepL 주소를 입력해 DeepL 홈페이지로 이동합니다

2. [무료 체험 시작하기] 버튼 클릭

화면 오른쪽 상단에 있는 무료 체험 시작하기 버튼을 클릭합니다

3. Starter 아래에 있는 [무료 체험] 버튼 클릭

Starter 아래에 있는 무료 체험 버튼을 클릭합니다

4. 이메일 주소 입력

새로운 계정을 생성하기 위해 이메일 주소와

5. 비밀번호 입력

비밀번호를 입력합니다

6. '만 14세 이상입니다.'에 체크

만 14세 이상입니다에 체크하고

7. [계속] 버튼 클릭

계속 버튼을 클릭합니다

8. 이름 입력

이어서 청구 정보를 입력합니다 이름과

9. 성 입력

성을 입력하고

10. 주소 입력

주소와

11. 우편번호 입력

우편번호

12. 시/군/구 입력

시 군 구를 입력합니다

13. 카드 번호 입력

이어서 결제 수단 정보를 입력합니다 카드 번호와

14. 만료일자 입력

만료 일자를 입력하고

15. 보안코드 입력

보안 코드를 입력합니다

16. [계속] 버튼 클릭

정보를 모두 입력했으면 계속 버튼을 클릭합니다

17. 약관에 동의

플랜 이용 조건을 확인하고 약관에 동의합니다

18. 이메일 수신에 동의

이메일 수신 여부를 선택하고

19. [지금 구매] 버튼 클릭

지금 구매 버튼을 클릭합니다

지금까지 DeepL에 회원 가입하고 무료 구독을 신청하는 방법을 살펴봤습니다
