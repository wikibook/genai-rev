OpenAI 키 발급하기

GPT for Sheets and Docs에서 챗GPT를 활용하려면 먼저 OpenAI API 키를 발급받아야 합니다. 이번 영상에서는 OpenAI API 키를 발급받는 방법을 살펴보겠습니다.

1. OpenAI API 홈페이지로 이동 (https://platform.openai.com/)

브라우저 주소창에 OpenAI API 주소를 입력해 OpenAI API 홈페이지로 이동합니다

2. [Log in] 클릭

화면 오른쪽 상단에 있는 로그인 버튼을 클릭합니다

3. OpenAI 계정에 로그인 [Google 계정으로 계속] 클릭

로그인 창이 나오면 OpenAI 계정으로 로그인합니다. 이 책에서는 구글 계정으로 로그인하기 위해 구글 계정으로 계속 버튼을 클릭했습니다.

4. [구글 계정] 클릭

로그인할 구글 계정을 선택합니다

5. [API keys] 클릭

로그인 후에 OpenAI API 화면이 나오면 마우스를 화면 왼쪽으로 가져갑니다. 화면 왼쪽에서 메뉴가 나오면 API keys 메뉴를 클릭합니다

6. [Create new secret key‍] 클릭

키를 발급받기 위해 화면 중앙에 있는 [Create new secret key] 버튼을 클릭합니다

7. 키 이름 입력

키를 발급하기 위한 창이 나오면 네임 아래쪽 입력 창에 키 이름을 입력하고

8. [Create secret key‍] 클릭

[Create secret key] 버튼을 클릭합니다

9. API 키 복사하여 안전한 곳에 보관

발급된 API 키는 창을 닫고 나면 다시는 볼 수 없습니다. API 키를 복사한 다음 안전한 곳에 잘 보관합니다

10. [Done‍] 클릭

API 키 발급을 마쳤으면 Done 버튼을 클릭합니다

11. [Usage] 클릭

API를 사용하면 사용량에 따라 비용이 부과됩니다. 이어서 사용량을 확인하는 방법을 살펴보겠습니다. 화면 왼쪽 메뉴에서 Usage 메뉴를 클릭합니다

12. 사용량 확인

Usage 화면에서는 API 사용량을 확인할 수 있습니다

지금까지 OpenAI API 키를 발급받고 사용량을 확인하는 방법을 살펴봤습니다
