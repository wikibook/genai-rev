DeepL 구독 해지하기

이번 영상에서는 DeepL의 유료 구독을 해지하는 방법을 살펴보겠습니다

1. 이메일 주소 오른쪽에 있는 버튼 클릭

화면 오른쪽 상단에 있는 프로필 아이콘을 클릭하거나 이메일 주소 오른쪽에 있는 버튼을 클릭합니다

2. [계정] 클릭

계정 메뉴를 클릭합니다

3. [구독 해지] 버튼 클릭

스크롤을 내린 다음 구독 취소 아래에 있는 구독 해지 버튼을 클릭합니다

4. [해지] 버튼 클릭

구독 취소 창이 나오면 해지 버튼을 클릭합니다

지금까지 디벨의 유료 구독을 해지하는 방법을 살펴봤습니다
