GPT for Sheets에 API 키 등록하기

이번 영상에서는 구글 스프레드시트에서 AI를 활용하는 방법을 살펴보겠습니다

1. 구글 드라이브로 이동 (https://drive.google.com)

주소창에 구글 드라이브 주소를 입력해 구글 드라이브로 이동합니다

2. [신규] 클릭

먼저 새로운 구글 스프레드시트를 생성하겠습니다. 왼쪽 메뉴에서 신규 버튼을 클릭하고

3. [Google 스프레드시트] 클릭

구글 스프레드시트 메뉴를 클릭합니다

4. [확장 프로그램] 클릭

새로운 구글 스프레드시트가 열리면 상단 메뉴에서 확장 프로그램을 클릭하고

5. [GPT for Sheets™ and Docs™] 클릭

GPT for Sheets and Docs 메뉴를 클릭합니다

6. [Set API key] 클릭

이어서 키를 설정하기 위해 Set API key 메뉴를 클릭합니다

7. OpenAI API 키 입력

오른쪽에 열린 GPT for Sheets and Docs 창에서 OpenAI APK 아래에 앞서 발급받은 OpenAI API 키를 입력합니다

8. [Save API key] 클릭

키를 입력했으면 아래에 있는 Save API key 버튼을 클릭합니다

9. [닫기] 클릭

마지막으로 닫기 버튼을 클릭해 줍니다

10. [확장 프로그램] 클릭

이어서 GPT for Sheets and Docs를 사용하려면 GPT Functions를 활성화해야 합니다. 다시 상단 메뉴에서 확장 프로그램을 클릭하고

11. 딸깍 하는 소리 "GPT for Sheets™ and Docs™►"

GPT for Sheets and Docs 메뉴를 클릭합니다

12. [Enable GPT functions] 클릭

이번에는 Enable GPT functions 메뉴를 클릭합니다

13. [확인] 클릭

GPT Functions를 활성화 할 것인지 묻는 창이 나오면 확인 버튼을 클릭합니다
